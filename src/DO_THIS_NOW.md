## Проблема

Сайт **иэо.рф** открывается БЕЗ СТИЛЕЙ (ошибки 404 на CSS/JS).

---

## ⚡ БЫСТРОЕ РЕШЕНИЕ

### Вариант 1: Если у вас есть SSH доступ

**Подключитесь к серверу:**

```bash
ssh root@217.114.15.80
```

**Выполните ЭТУ команду (скопируйте всё):**

```bash
cd /var/www/ivo && rm -rf build dist && npm run build && chown -R www-data:www-data /var/www/ivo/build/ && chmod -R 755 /var/www/ivo/build/ && systemctl restart nginx && echo "✅ ГОТОВО! Откройте http://иэо.рф и нажмите Ctrl+F5"
```

**В браузере:**
1. Откройте: **http://иэо.рф**
2. Нажмите: **Ctrl + F5**

✅ **ГОТОВО!**

---

Если на сервере старые файлы, сначала загрузите новые:

## 🔍 Если всё ещё не работает

Проверьте в консоли браузера (F12 → Network):

- ❌ `404` на CSS? → Файлы не собраны
- ❌ `404` на JS? → Файлы не собраны
- ✅ `200 OK`? → Всё работает!

**Решение:** Выполните команду из Варианта 1 ещё раз.

---

## 📞 Техническая информация

**Причина проблемы:**
- CSS/JS файлы не собраны или не на месте
- Nginx не может их найти

**Решение:**
- Пересобрать проект: `npm run build`
- Установить права: `chown` и `chmod`
- Перезапустить nginx: `systemctl restart nginx`
