## ✅ РЕШЕНИЕ (выберите один вариант)

### 🚀 ВАРИАНТ 1: Локальная сборка + загрузка (РЕКОМЕНДУЕТСЯ)

#### Windows:
```batch
# 1. Соберите проект
build-local.bat

# 2. Загрузите через WinSCP/FileZilla
# (инструкции появятся после сборки)
```

#### Linux/Mac:
```bash
# 1. Соберите проект
chmod +x build-local.sh
./build-local.sh

# 2. Загрузите на сервер
chmod +x deploy-build.sh
./deploy-build.sh
```

**Или вручную:**
```bash
# Сборка
npm install
npm run build

# Загрузка
scp -r build/* root@217.114.15.80:/var/www/ivo/build/

# Права доступа (на сервере)
ssh root@217.114.15.80
chown -R www-data:www-data /var/www/ivo/build/
chmod -R 755 /var/www/ivo/build/
systemctl restart nginx
```

## 🆘 Если не работает

### Проблема: Всё равно 404 на CSS/JS

**Диагностика:**
```bash
# На сервере
ls -la /var/www/ivo/build/
ls -la /var/www/ivo/build/assets/
cat /var/www/ivo/build/index.html | grep assets
```

**Решение:**
Удалите build полностью и пересоберите:
```bash
ssh root@217.114.15.80
cd /var/www/ivo
rm -rf build
npm run build
```

### Проблема: npm run build не работает на сервере

**Причина:** Нет исходников или зависимостей

**Решение:**
1. Соберите локально (`build-local.bat` или `build-local.sh`)
2. Загрузите папку `build/` на сервер
