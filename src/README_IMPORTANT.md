# 🚨 ИСПРАВЛЕНИЕ 404 ОШИБКИ

## Что случилось?

Сайт **иэо.рф** открывается, но **CSS/JS не загружаются** (ошибка 404).

---

## ⚡ БЫСТРОЕ РЕШЕНИЕ

### 1. Подключитесь к серверу:

```bash
ssh root@217.114.15.80
```

### 2. Выполните ЭТУ команду (скопируйте всё):

```bash
cd /var/www/ivo && rm -rf build dist && npm run build && chown -R www-data:www-data /var/www/ivo/build/ && chmod -R 755 /var/www/ivo/build/ && systemctl restart nginx && echo "✅ ГОТОВО! Откройте http://иэо.рф и нажмите Ctrl+F5"
```

### 3. В браузере:

1. Откройте: **http://иэо.рф**
2. Нажмите: **Ctrl + F5** (жёсткая перезагрузка)
