## 🆘 Решение проблем

### Ошибка: 404 на CSS/JS файлы

**Причина:** Неправильные хеши файлов или файлы не на месте

**Решение:**
```bash
# На сервере
cd /var/www/ivo
rm -rf build
npm run build
chown -R www-data:www-data /var/www/ivo/build/
systemctl restart nginx
```

### Ошибка: Стили не применяются

**Причина:** Кеш браузера

**Решение:**
- Ctrl+Shift+R (жёсткая перезагрузка)
- Или очистите кеш в настройках браузера

### Ошибка: Страницы не открываются (404)

**Причина:** Неправильная конфигурация nginx

**Решение:**
```bash
# Проверьте конфигурацию
nginx -t

# Если есть ошибки, исправьте конфиг
nano /etc/nginx/sites-available/ivo

# Перезапустите nginx
systemctl restart nginx
```
