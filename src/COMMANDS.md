# ⚡ Шпаргалка по командам - Проект ИЭО

## 🚀 Основные команды

### Первый запуск
```bash
# 1. Установить зависимости (один раз)
npm install

# 2. Запустить сервер разработки
npm run dev

# 3. Открыть в браузере: http://localhost:5173/
```

---

## 📝 Все команды

| Команда | Что делает | Когда использовать |
|---------|------------|-------------------|
| `npm install` | Устанавливает все зависимости | При первом запуске или после git pull |
| `npm run dev` | Запускает сервер разработки | Для локальной разработки |
| `npm run build` | Собирает production версию | Перед загрузкой на Beget |
| `npm run preview` | Просмотр production сборки | Проверка перед деплоем |

---

## 💻 Разработка

### Запуск локального сервера
```bash
npm run dev
```
**Результат:** Сервер запущен на http://localhost:5173/

### Запуск на другом порту
```bash
npm run dev -- --port 3000
```

### Доступ с телефона (в локальной сети)
```bash
npm run dev -- --host
```
**Результат:** Показывает IP-адрес для доступа с других устройств

---

## 🏗️ Сборка

### Production сборка
```bash
npm run build
```
**Результат:** Создаётся папка `dist/` с готовым сайтом

### Проверка production сборки
```bash
npm run preview
```
**Результат:** Запускается на http://localhost:4173/

## 🔧 Обслуживание

### Переустановка зависимостей
```bash
rm -rf node_modules package-lock.json
npm install
```

### Очистка кеша Vite
```bash
rm -rf node_modules/.vite
npm run dev
```

### Обновление зависимостей
```bash
npm update
```

## 📦 Деплой на Beget

### Полный цикл деплоя
```bash
# 1. Собрать проект
npm run build

# 2. Проверить локально
npm run preview

# 3. Загрузить dist/ на Beget через FTP
# 4. Загрузить .htaccess в public_html/
```

## ⚠️ Важные заметки

1. **Всегда** запускайте `npm install` после скачивания проекта
2. **Не загружайте** папку `node_modules` на Beget
3. **Загружайте** только содержимое папки `dist/`
4. **Обязательно** добавьте `.htaccess` на сервер

## 🆘 Решение проблем

| Проблема | Решение |
|----------|---------|
| Порт занят | `npm run dev -- --port 3000` |
| Ошибки зависимостей | `rm -rf node_modules && npm install` |
| Изменения не видны | Ctrl+C, затем `npm run dev` |
| Белый экран | Проверьте консоль (F12) |
