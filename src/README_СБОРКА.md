## 🔧 Процесс сборки

### 1. Локальная разработка

```bash
npm run dev
```

Откроется локальный сервер на http://localhost:5173

### 2. Сборка для production

```bash
npm install       # Установить зависимости
npm run build     # Собрать проект
```

**Результат:** Создастся папка `build/` с оптимизированными файлами.

### 3. Деплой на сервер

Выберите один из способов:

**A. Автоматически (Linux/Mac):**
```bash
./deploy-build.sh
```

**B. Вручную через SCP:**
```bash
scp -r build/* root@217.114.15.80:/var/www/ivo/build/
ssh root@217.114.15.80 "chown -R www-data:www-data /var/www/ivo/build/ && chmod -R 755 /var/www/ivo/build/ && systemctl restart nginx"
```

**C. Через WinSCP/FileZilla:**
1. Подключиться к `217.114.15.80`
2. Удалить `/var/www/ivo/build/`
3. Загрузить новую папку `build/`
4. Установить права через SSH (см. команду B)

## 🆘 Решение проблем

### Проблема: Браузер запрашивает файлы с неправильными хешами

**Причина:** На сервере старая сборка

**Решение:**
```bash
# Пересоберите локально
npm run build

# Загрузите на сервер
scp -r build/* root@217.114.15.80:/var/www/ivo/build/

# Установите права
ssh root@217.114.15.80 "chown -R www-data:www-data /var/www/ivo/build/ && chmod -R 755 /var/www/ivo/build/ && systemctl restart nginx"
```

### Проблема: 404 на CSS/JS файлы

**Диагностика:**
```bash
ssh root@217.114.15.80 "ls -la /var/www/ivo/build/assets/"
```

**Решение:**
См. решение выше.

### Проблема: Стили не применяются

**Причина:** Кеш браузера

**Решение:**
- Ctrl+Shift+R (жёсткая перезагрузка)
- Очистить кеш в настройках браузера
- Открыть в режиме инкогнито

### Проблема: npm run build падает с ошибкой

**Возможные причины:**
1. Не установлены зависимости → `npm install`
2. Нет места на диске → освободить место
3. Ошибки в коде → проверить `npm run dev`
