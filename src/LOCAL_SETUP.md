# 💻 Запуск проекта ИЭО на локальном компьютере

## 📋 Требования

- **Node.js** версии 18 или выше
- **npm** (устанавливается вместе с Node.js)
- Любой современный браузер (Chrome, Firefox, Edge)

---

## 🚀 Быстрый старт

### 1. Установите Node.js (если ещё не установлен)

**Windows:**
1. Скачайте установщик: https://nodejs.org/
2. Выберите версию **LTS** (рекомендуется)
3. Запустите установщик и следуйте инструкциям
4. Перезагрузите командную строку

**macOS:**
```bash
# Через Homebrew
brew install node
```

**Linux (Ubuntu/Debian):**
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**Проверка установки:**
```bash
node --version  # Должно показать v18.x.x или выше
npm --version   # Должно показать 9.x.x или выше
```

---

## 📦 Установка зависимостей

Откройте терминал/командную строку в папке проекта и выполните:

```bash
npm install
```

Это установит все необходимые библиотеки:
- React и React DOM
- Lucide Icons
- Radix UI компоненты
- Tailwind CSS
- Vite (сборщик)
- TypeScript

**Время установки:** 1-3 минуты (зависит от скорости интернета)

---

## ▶️ Запуск сервера разработки

```bash
npm run dev
```

После запуска вы увидите сообщение:

```
  VITE v5.4.0  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

**Откройте в браузере:** http://localhost:5173/

---

## 🎨 Режимы работы

### Режим разработки (Development)
```bash
npm run dev
```
- **Hot Reload** - изменения отображаются мгновенно
- Подробные сообщения об ошибках
- Source maps для отладки
- Запускается на http://localhost:5173/

### Сборка для production
```bash
npm run build
```
- Создаёт оптимизированную версию в папке `dist/`
- Минификация кода
- Оптимизация изображений
- Готово для загрузки на Beget

### Предварительный просмотр production сборки
```bash
npm run preview
```
- Запускает локальный сервер с production версией
- Позволяет протестировать финальную сборку
- Запускается на http://localhost:4173/

## 🛠️ Разработка

### Внесение изменений

1. Откройте файл в любом редакторе кода (VS Code рекомендуется)
2. Внесите изменения
3. Сохраните файл (Ctrl+S / Cmd+S)
4. Браузер автоматически обновится с изменениями

### Добавление новой страницы

1. Создайте файл в `components/`, например `NewPage.tsx`
2. Добавьте маршрут в `App.tsx`
3. Добавьте кнопку в `Header.tsx`

### Изменение стилей

- Глобальные стили: `styles/globals.css`
- Tailwind классы: прямо в компонентах
- Цвета триколора уже настроены в `globals.css`

## 🔧 Решение проблем

### Порт 5173 уже занят

```bash
# Укажите другой порт
npm run dev -- --port 3000
```

### Ошибка "Cannot find module"

```bash
# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install
```

### Изменения не применяются

1. Остановите сервер (Ctrl+C)
2. Очистите кеш: `rm -rf node_modules/.vite`
3. Запустите снова: `npm run dev`

### Ошибки TypeScript

```bash
# Проверьте типы
npx tsc --noEmit
```

## 📊 Что происходит при запуске

1. **`npm run dev`** запускает Vite сервер
2. Vite компилирует TypeScript → JavaScript
3. Vite обрабатывает Tailwind CSS
4. Сервер запускается на localhost:5173
5. Открывается страница в браузере
6. Hot Module Replacement отслеживает изменения
