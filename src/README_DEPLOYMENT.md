## 🔧 Что было исправлено?

### Проблема
Сайт на **иэо.рф** отображался без цветов - только текст на белом фоне.

### Причина
В файле `/styles/globals.css` отсутствовала обязательная директива для Tailwind CSS v4:
```css
@import "tailwindcss";
```

### Решение
Добавлена первая строка в `globals.css`:
```css
@import "tailwindcss";

@custom-variant dark (&:is(.dark *));
...
```

Теперь Tailwind корректно компилируется и генерирует все стили триколора.

## 🎯 Быстрый старт

### 1. Соберите проект

```bash
npm install
npm run build
```

Результат: папка `dist/` с готовым сайтом

### 2. Загрузите на Beget

**Вариант А: Файловый менеджер**
1. Войдите на https://beget.com
2. Откройте "Файловый менеджер"
3. Перейдите в `иэо.рф/public_html/`
4. Загрузите всё из `dist/`:
   - `index.html`
   - `favicon.svg`
   - папку `assets/`

**Вариант Б: FTP (FileZilla)**
1. Подключитесь к FTP Beget
2. Загрузите содержимое `dist/` в `public_html/`

### 3. Проверьте

Откройте: https://иэо.рф
