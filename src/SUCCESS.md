## 🚨 Если что-то не работает

### Сайт не открывается?

```bash
# На сервере проверьте nginx:
systemctl status nginx

# Перезапустите:
systemctl restart nginx

# Проверьте порты:
netstat -tlnp | grep :80
```

### Нет цветов?

```bash
# Проверьте файл CSS:
ls -la /var/www/ivo/build/assets/

# Должен быть файл: index-B5saN5aj.css (~24 KB)

# Проверьте содержимое (первые 20 строк):
head -20 /var/www/ivo/build/assets/index-B5saN5aj.css
```

Должны увидеть CSS правила с цветами.

### Очистите кеш браузера:

```
Ctrl + Shift + Delete
или
Ctrl + F5 (жесткая перезагрузка)
```
