## 🛠️ Скрипты автоматизации

### Windows (BAT файлы)

| Файл | Команда запуска | Описание |
|------|----------------|----------|
| **build-local.bat** | `build-local.bat` | Сборка проекта на Windows |
| **deploy-build.bat** | `deploy-build.bat` | Инструкции по загрузке для Windows |

### Linux/Mac (SH файлы)

| Файл | Команда запуска | Описание |
|------|----------------|----------|
| **build-local.sh** | `chmod +x build-local.sh && ./build-local.sh` | Сборка проекта на Linux/Mac |
| **deploy-build.sh** | `chmod +x deploy-build.sh && ./deploy-build.sh` | Автоматическая загрузка на сервер |

**Примечание:** Файлы `.sh` нужно сделать исполняемыми командой `chmod +x` перед первым запуском.

## 🔑 Ключевые команды

### Одна команда для всего (Linux/Mac)

```bash
npm run build && scp -r build/* root@217.114.15.80:/var/www/ivo/build/ && ssh root@217.114.15.80 "chown -R www-data:www-data /var/www/ivo/build/ && chmod -R 755 /var/www/ivo/build/ && systemctl restart nginx"
```

### Сборка на сервере (если исходники там)

```bash
ssh root@217.114.15.80 "cd /var/www/ivo && rm -rf build && npm install && npm run build && chown -R www-data:www-data build/ && chmod -R 755 build/ && systemctl restart nginx"
```

Подробнее см. `КОПИРОВАТЬ_ЭТИ_КОМАНДЫ.txt`

## 💡 Советы

### Если вы впервые работаете с проектом

1. Начните с `БЫСТРОЕ_РЕШЕНИЕ.md`
2. Если используете Windows - откройте `ИНСТРУКЦИЯ_WINSCP.md`
3. Держите под рукой `КОМАНДЫ.txt` для быстрого доступа к командам

### Если вы опытный разработчик

1. Откройте `КОПИРОВАТЬ_ЭТИ_КОМАНДЫ.txt`
2. Скопируйте нужную команду и выполните
3. Готово!

### Если возникли проблемы

1. Проверьте раздел "Решение проблем" в `СБОРКА_И_ДЕПЛОЙ.md`
2. Посмотрите диагностические команды в `КОПИРОВАТЬ_ЭТИ_КОМАНДЫ.txt`
3. Проверьте логи nginx:
   ```bash
   ssh root@217.114.15.80 "tail -n 50 /var/log/nginx/error.log"
   ```
