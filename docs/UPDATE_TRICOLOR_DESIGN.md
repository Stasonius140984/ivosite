## 🚀 КАК ОБНОВИТЬ НА СЕРВЕРЕ

### Вариант 1: Загрузить все файлы заново (РЕКОМЕНДУЕТСЯ)

**На вашем компьютере:**

1. Скачайте ВСЕ файлы из этого проекта
2. Упакуйте в zip архив
3. Загрузите на сервер через SFTP (WinSCP, FileZilla)

**На сервере:**

```bash
# Подключитесь
ssh root@217.114.15.80

# Остановите старые файлы (сделайте бэкап)
mv /var/www/ivo /var/www/ivo_backup_old

# Распакуйте новые файлы
unzip /путь/к/новому/архиву.zip -d /var/www/ivo

# Установите зависимости
cd /var/www/ivo
npm install

# Соберите проект
npm run build

# Установите права
chown -R www-data:www-data /var/www/ivo/build/
chmod -R 755 /var/www/ivo/build/
find /var/www/ivo/build/ -type f -exec chmod 644 {} \;

# Перезапустите nginx
systemctl restart nginx

# Проверьте
curl -I http://217.114.15.80
```

## 🎨 Цветовая схема

### Официальные цвета российского флага:

| Цвет | Hex код | Где используется |
|------|---------|------------------|
| **Белый** | `#FFFFFF` | Основной фон контента |
| **Синий** | `#0039A6` | Header, границы карточек, иконки, заголовки |
| **Красный** | `#D52B1E` | Акценты, границы, важные элементы |

### Применение:

```css
/* Синий флага */
--tricolor-blue: #0039A6;
bg-[#0039A6]
text-[#0039A6]
border-[#0039A6]

/* Красный флага */
--tricolor-red: #D52B1E;
bg-[#D52B1E]
text-[#D52B1E]
border-[#D52B1E]
```
