### 2. Проверьте папку `dist/`

Должна содержать:
```
dist/
├── index.html          ✓ HTML файл
├── favicon.svg         ✓ Иконка
└── assets/
    ├── index-XXXXX.css ✓ CSS со стилями (НОВЫЙ!)
    ├── index-XXXXX.js  ✓ JavaScript
    └── ...
```

---

### 3. Войдите в Beget

1. Откройте: https://beget.com
2. Введите логин и пароль
3. Перейдите в "Файловый менеджер"

---

### 4. Подготовьте папку на сервере

**Путь:** `иэо.рф/public_html/` (или `xn--d1a4b.xn--p1ai/public_html/`)

**Удалите старые файлы:**
- ❌ Старый `index.html`
- ❌ Старую папку `assets/`
- ✅ Оставьте `.htaccess` (НЕ удаляйте!)
- ✅ Оставьте `favicon.svg` (можно обновить)

---

### 5. Загрузите новые файлы

**Из папки `dist/` загрузите:**
1. ✅ `index.html` → в `public_html/`
2. ✅ `favicon.svg` → в `public_html/`
3. ✅ Папку `assets/` → в `public_html/`

**Итоговая структура на сервере:**
```
public_html/
├── .htaccess           ← Был загружен ранее
├── index.html          ← НОВЫЙ (сейчас)
├── favicon.svg         ← Обновленный
└── assets/             ← НОВАЯ папка
    ├── index-XXXXX.css ← НОВЫЙ CSS!
    ├── index-XXXXX.js
    └── ...
```

---

### 6. Проверьте права доступа

В файловом менеджере Beget:
- **Файлы** (index.html, .htaccess, и т.д.): `644`
- **Папки** (assets/, public_html/): `755`

Обычно права выставляются автоматически, но проверьте!

## 🔍 Диагностика проблем:

### Если сайт всё ещё без цветов:

**1. Откройте консоль браузера:**
- Нажмите `F12`
- Вкладка "Console"
- Проверьте ошибки (красные строки)

**2. Проверьте загрузку CSS:**
- В консоли откройте вкладку "Network"
- Обновите страницу (`F5`)
- Найдите файл `index-XXXXX.css`
- Статус должен быть `200 OK`
- Если `404 Not Found` → CSS не загружен на сервер

**3. Проверьте содержимое CSS:**
- В консоли вкладка "Sources"
- Откройте `assets/index-XXXXX.css`
- Должны быть Tailwind классы (`.bg-blue-900`, `.text-white`, и т.д.)
